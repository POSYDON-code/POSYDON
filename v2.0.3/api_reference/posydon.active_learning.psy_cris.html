Bases: object

For managing data sets used for classification and regression.

Reads tables of simulation data where a single row represents one simulation. Each column in a row represents different inputs (initial conditions) and outputs (result, continuous variables). If using multiple files, each file is assumed to have the same columns. You may also directly load a pandas DataFrame instead of reading in files.

Example data structure expected in files or pandas DataFrame:

0 input_1 input_2 outcome output_1 output_2 output_3 ‚Ä¶ 1 1.5 2.6 ‚ÄúA‚Äù 100 0.19 - ‚Ä¶ 2 1.5 3.0 ‚ÄúB‚Äù - - - ‚Ä¶ 3 2.0 2.6 ‚ÄúC‚Äù - - 6 ‚Ä¶ ‚Ä¶

The above table has dashes ‚Äò-‚Äô in output columns to indicate NaN values. You may have a similar structure if different classes have fundamentally different outputs.

Initialize the TableData instance.

Parameters:

  • table_paths (list) ‚Äì List of file paths to read in as data. if None, a pandas DataFrame is used instead

  • input_cols (list) ‚Äì List of names of the columns which will be considered ‚Äòinput‚Äô.

  • output_cols (list) ‚Äì List of names of the columns which will be considered ‚Äòoutput‚Äô. This should include the class column name.

  • class_col_name (str) ‚Äì Name of column which contains classification data.

  • my_DataFrame (pandas DataFrame, optional) ‚Äì If given, use this instead of reading files.

  • omit_vals (list, optional) ‚Äì Numerical values that you wish to omit from the entire data set. If a row contains the value, the entire row is removed. (For example you may want to omit all rows if they contain ‚Äú-1‚Äù or ‚Äúfailed‚Äù.)

  • omit_cols (list, optional) ‚Äì Column names that you wish to omit from the data set.

  • subset_interval (array, optional) ‚Äì Use some subset of the data files being loaded in. An array with integers indicating the rows that will be kept.

  • my_colors (list, optional) ‚Äì Colors to use for classification plots.

  • n_neighbors (list, optional) ‚Äì List of integers that set the number of neighbors to use to calculate average distances. (default None)

  • neighbor (instance of sklearn.neighbors.NearestNeighbors, optional) ‚Äì To use for average distances. See function ‚Äòcalc_avg_dist()‚Äô.

  • undefined_p_change_val (optional, float) ‚Äì Sets the undefined value used when calculating percent change fails due to zero values in the output data. Default uses nan.

  • verbose (bool, optional) ‚Äì Print statements with extra info.

  • read_csv_kwargs (dict, optional) ‚Äì Kwargs passed to the pandas function ‚Äòread_csv()‚Äô.

Get binary mapping (0 or 1) of the class data for each unique classification. For each classification, a value of 1 is given if the class data matches that classification. If they do not match, then a value of 0 is given.
