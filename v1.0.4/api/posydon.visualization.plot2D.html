Plotting class for 2D (MESA) psygrids.

The 2D visualization plotting class allows to plot 2D slices of PsyGrid objects. The PsyGrid object is composed of 2D/3D/4D MESA grid run with POSYDON and post processed with the psygrid object into an h5 file.

Plotting class for 2D (MESA) grids.

Read a PsyGrid object and plot a 2D slice of x vs y.

Parameters

  • psygrid (object) ‚Äì PsyGrid object containing a 2D/3D/4D MESA grid.

  • x_var_str (str) ‚Äì String of the initial value to plot on the x axis. Allowed strings are psygrid.initial_values.dtype.names.

  • y_var_str (str) ‚Äì String of the initial value to plot on the y axis. Allowed strings are psygrid.initial_values.dtype.names.

  • z_var_str (str) ‚Äì String of the initial value to plot on the z axis (displayed as a color). Allowed strings are psygrid.final_values.dtype.names, psygrid.history1.dtype.names or psygrid.history2.dtype.names depending on ‚Äúselected_star_history_for_z_var‚Äù value, and psygrid.binary_history.dtype.names.

  • selected_star_history_for_z_var (int) ‚Äì Accepted valuess: 1 or 2. In case z_var_str is an attribute of history1 or history2, then selected_star_history_for_z_var determines which of the two to select.

  • termination_flag (str) ‚Äì Termination flag to display, allowed values are: ‚Äútermination_flag_1‚Äù, ‚Äútermination_flag_2‚Äù, ‚Äútermination_flag_3‚Äù, ‚Äútermination_flag_4‚Äù, ‚Äúall‚Äù.

  • grid_3D (bool) ‚Äì If True, the psygrid object is a 3D grid and needs to be sliced.

  • slice_3D_var_str (str) ‚Äì Variable along which the 3D space will be sliced. Allowed values are psygrid.initial_values.dtype.names.

  • slice_3D_var_range (tuple) ‚Äì Range between which you want to slice the variable slice_3D_var_str e.g., (2.5,3.).

  • grid_4D (bool) ‚Äì If True, the psygrid object is a 4D grid and needs to be sliced.

  • slice_4D_var_str (str) ‚Äì Variable along which the 4D space will be sliced. Allowed values are psygrid.initial_values.dtype.names.

  • slice_4D_var_range (tuple) ‚Äì Range between which you want to slice the variable slice_4D_var_str e.g., (2.5,3.).

  • extra_grid (object or array of objects) ‚Äì If subset of the grid was rerun a or an extention was added, one can overlay the new psygrid by passing it here.

  • slice_at_RLO (bool) ‚Äì If True, the object plots the tracks until onset of Roche Lobe overflow.

  • MARKERS_COLORS_LEGENDS (dict) ‚Äì Each termination flag is associated with a marker shape, size, color and label (cf. MARKERS_COLORS_LEGENDS in plot_defaults.py).

  • DEFAULT_LABELS (dict) ‚Äì Each varaible is associated to an axis label. (cf. DEFAULT_LABELS in plot_defaults.py).

  • verbose (bool) ‚Äì If True, the object reports by printing to standard output.

  • **kwargs (dict) ‚Äì Dictionary containing extra visualisation options (cf. PLOT_PROPERTIES in plot_defaults.py.
